# Current weather app

Disclaimer: this is a simplified version of a weather app, it is not very accurate.

Uses OpenStreetMap and open-meteo's public API.

## Usage


1. Find place of your interest dragging the map;
2. Click on the map point you want to see weather;
3. A loading screen will be shown;
4. Finally, will be shown weather screen (with some weather data).

Alternative way:
1. Open drawer (button in top-left of the screen);
2. Choose from one of pre-defined places;
3. Click on "Go to" button.
